# 均线（MA）单因子回测结论

> 基于 20 轮手动迭代调参（2026-03-05）

## 1. 回测设置

- **数据**：BTCUSDT 15 分钟 K 线，约 10 万根（1042 天）
- **流程**：运行 `go run ./cmd/autotest` → 分析 Top 5 → 更新 `case_ma.go` → 重复 20 次
- **目标**：预测下一根 K 线涨跌方向，以信号准确率为指标

## 2. 主要结论

### 2.1 最优参数

| 项目 | 值 |
|------|-----|
| **最佳策略** | MA_1_7（短期 1，长期 7） |
| **信号准确率** | 52.84% |
| **信号方向** | 短期均线 < 长期均线 → 看涨；短期 > 长期 → 看跌 |

### 2.2 信号方向与实现

- **原始逻辑**（多头排列）：短期 > 长期 → 看涨
- **回测结论**：在上述数据上，**反向信号**预测效果更好
- **实现调整**：已在 `FactorMA` 中固定为正向信号 —— 短期 < 长期 → 看涨，因子直接输出该方向

### 2.3 Top 5 均线组合（按准确率排序）

| 排名 | 组合 | 准确率 |
|------|------|--------|
| 1 | MA_1_7 | 52.84% |
| 2 | MA_1_8 | 52.78% |
| 3 | MA_1_6 | 52.77% |
| 4 | MA_1_9 | 52.73% |
| 5 | MA_1_5 | 52.71% |

短周期（1–10）表现普遍优于长周期（20、30、60 等）。

### 2.4 单因子与权重

- **单因子**：方向由因子实现决定，权重只改变方向或缩放，无优化意义
- **多因子**：权重用于调节各因子贡献度，才有优化价值
- **当前做法**：`FactorMA` 输出正向信号，多因子场景下权重仅负责调节贡献度

## 3. 20 轮迭代演进

| 轮次 | 最佳策略 | 最佳准确率 |
|------|----------|------------|
| 1 | MA_5_10 | 51.55% |
| 2 | MA_3_5 | 52.23% |
| 3 | MA_2_3 | 52.66% |
| 4–20 | MA_1_7 | 52.84% |

探索路径：长周期 → 短周期 → 超短周期（1, 7）。
