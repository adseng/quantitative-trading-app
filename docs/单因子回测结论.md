## 二、因子方向修正说明

单因子场景下权重无意义，只有多因子组合时权重才用于调节贡献度。基于回测结论，以下因子已修改为输出正向信号：

| 因子 | 原逻辑 | 修正后 |
|------|--------|--------|
| MA | 短期>长期→看涨 | 短期<长期→看涨 |
| Trend | 净差>0→看涨 | 净差<0→看涨 |
| MACD | MACD>信号线→看涨 | MACD<信号线→看涨 |
| PriceVsMA | 价>SMA→看涨 | 价<SMA→看涨 |
| ATR | 放大+涨→看涨 | 放大+跌→看涨 |
| Volume | 放量涨→看涨 | 放量跌→看涨 |
| Session | 欧美盘涨→看涨 | 欧美盘跌→看涨 |
| Breakout | 突破最高→看涨 | 跌破最低→看涨 |
| MACross | 金叉→看涨 | 死叉→看涨 |
| Boll | 下轨→看涨、上轨→看跌 | 无需修正（原生正确） |
| RSI | 超卖→看涨、超买→看跌 | 无需修正（原生正确） |
