## 二、结论

### 1. 单因子 vs 多因子

- **单因子 Boll P13 M2.4** 全局最佳（59.71%），信号少（3.8%）
- **单因子 Break P15** 次优（57.65%），信号率 10.7%
- **双因子 Bo+Br** 最优（58.06%），在双因子中表现最好，略逊于单因子 Boll，但信号率提升至 11%

### 2. 双因子组合排序（按准确率）

1. Bo+Br 58.06% — Boll P13 M2.2 + Break P20
2. Bo+R 56.69% — Boll P13 M2.0 + RSI P9 80/20
3. R+Br 56.25% — RSI P7 80/20 + Break P12
4. Bo+A 54.15%
5. Br+A 54.08%
6. Bo+M / Bo+Pv / Bo+T 约 52~53%

### 3. 权重与参数

- **Bo+Br**：Boll P13 M2.2、Break P20，权重 1:1 已较优
- **Bo+R**：Boll 偏好 M2.0、RSI 用 80/20 或 75/25 均可
- 多数双因子组合下，权重 1:1 或 2:1 差异不大，主要影响信号强度

## 三、推荐配置

### 优先：单因子 Boll
```
Boll: Period=13, Multiplier=2.4, Weight=1
```

### 备选：双因子 Bo+Br（略多信号）
```
Boll: Period=13, Multiplier=2.2, Weight=1
Breakout: Period=20, Weight=1
```

### 五因子 Bo+R+Br+A+Volume（五因子最优，53.86%）
```
RSI: P7 80/20, Weight=1
Boll: Period=13, Multiplier=2.0, Weight=1
Breakout: Period=12, Weight=1
ATR: Period=20, Weight=1
Volume: Period=10, Weight=0.25
```

### 四因子 Bo+R+Br+A（若需更高信号率）
```
RSI: P7 80/20, Weight=1
Boll: Period=13, Multiplier=2.0, Weight=1
Breakout: Period=12, Weight=1
ATR: Period=20, Weight=1
```

### 三因子 RSI+Boll+Break（信号率 24.6%，推荐）
```
RSI: P7 80/20, Weight=1
Boll: Period=13, Multiplier=2.0, Weight=1
Breakout: Period=15, Weight=1
```

### 单因子 Break（信号更多）
```
Breakout: Period=15, Weight=1
```

## 四、四因子结论（基于 Bo+R+Br 叠加第 4 因子）

| 第4因子 | 最佳准确率 | 信号率 | 结论 |
|---------|------------|--------|------|
| ATR | 54.47% | 47.5% | 最优四因子，波动过滤有效 |
| Session | 53.09% | 75.0% | 时段过滤，略优于 T/M |
| Trend | 53.03% | 69.8% | 趋势确认 |
| MA | 52.85% | 100% | 信号过滥，准确率下降 |

**结论**：三因子 Bo+R+Br(56.41%) 仍优于所有四因子；若需更高信号率，优先考虑加 ATR(54.47%)。

## 六、五因子结论（基于 Bo+R+Br+A 叠加第5因子）

| 第5因子 | 最佳准确率 | 信号率 | 最优参数 | 结论 |
|---------|------------|--------|----------|------|
| Session | 52.92% | 81.4% | R7+Bo12M2.0+Br12+A20+Sess0.25 | 加 Session 后准确率明显下降 |
| Trend | 53.12% | 79.1% | R7+Bo13M2.0+Br15+A20+T6_w0.25 | 略优于 Sess，仍低于四因子 |
| MA | 52.78% | 100% | R5+Bo13M2.0+Br12+A20+M7_w0.25 | 信号过滥，准确率低 |
| PriceVsMA | 52.75% | 100% | R5+Bo13M2.0+Br12+A20+Pv8_w0.25 | 与 MA 类似 |

**结论**：五因子排序 **Volume(53.86%)** > Trend(53.12%) > Sess(52.92%) > MA(52.78%) > Pv(52.75%)。Volume 叠加后准确率最接近四因子(54.47%)，信号率 54.5% 适中；其余因子叠加后均明显下降。
