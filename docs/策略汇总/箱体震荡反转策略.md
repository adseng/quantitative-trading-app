# 箱体震荡反转策略

## 1. 策略定位
- 目标：专门做横盘震荡，不参与趋势追涨杀跌。
- 核心前提：当前市场不是普通随机横盘，而是经历过一段明显的大幅上涨或大幅下跌后，进入缩量整理阶段。
- 收益来源：在箱体上沿做空、箱体下沿做多，吃价格回到箱体中轴或对侧边界的回归收益。

## 2. 核心思路
- 先识别一段明显的冲击行情，确认市场刚经历过大幅涨跌。
- 再识别冲击后的缩量横盘整理，而不是只看最近固定 `20~40` 根高低点。
- 横盘成立后，只在箱体边界附近寻找反转确认，不在箱体中间追单。
- 多空对称：上沿找做空机会，下沿找做多机会。

## 3. 横盘识别规则
### 冲击段成立
- 在进入横盘前，必须先存在一段明显的单边冲击，可为上涨或下跌。
- 冲击段的总涨跌幅必须大于最近常态波动的若干倍，例如：
- 最近 `impulseLookback` 根内累计涨跌幅绝对值大于 `minImpulsePercent`。
- 或者冲击段振幅大于最近 `ATR` 均值的 `minImpulseATRRatio` 倍。
- 冲击段期间的成交量应明显放大，代表这不是普通噪音波动。

### 缩量整理成立
- 冲击结束后，价格进入一段收敛整理区。
- 这段整理期内，最高价与最低价形成清晰箱体，箱体宽度不得过宽，也不能过窄。
- 整理期内成交量均值应低于冲击段成交量均值，满足 `consolidationVolumeRatio` 上限。
- 整理期内 ATR 均值应明显低于冲击段 ATR 均值，满足 `consolidationATRRatio` 上限。
- 整理期内快均线和慢均线开始走平，斜率下降，避免把趋势中继误判成震荡。

### 箱体有效性
- 箱体至少被上下边界多次测试，而不是只出现一次触碰。
- 上沿和下沿都应存在至少 `minBoundaryTouches` 次有效试探。
- 若价格实体连续收在箱体外，视为横盘失效，不再按震荡策略处理。

## 4. 多头规则
### 前提
- 当前已识别为“冲击后缩量横盘”。
- 当前价格接近箱体下沿，距离不超过 `edgeTolerancePercent`。

### 反转确认
- 触碰或刺破下沿后，出现止跌确认 K 线。
- 确认方式可以是以下组合：
- 当前 K 线收阳。
- 下影线明显长于实体，说明有承接。
- 收盘重新回到箱体内部。
- 成交量不需要放大到冲击段水平，但不能出现放量破位。

### 入场与风控
- 入场：确认 K 线后，下一根 K 线开盘价做多。
- 止损：放在箱体下沿之外，再加 `ATR * stopATRMultiplier` 缓冲。
- 第一止盈：箱体中轴。
- 第二止盈：箱体上沿。
- 首版回测可先简化为固定目标：
- 保守版：到中轴止盈。
- 进取版：到对侧边界止盈。

## 5. 空头规则
### 前提
- 当前已识别为“冲击后缩量横盘”。
- 当前价格接近箱体上沿，距离不超过 `edgeTolerancePercent`。

### 反转确认
- 触碰或刺破上沿后，出现滞涨确认 K 线。
- 确认方式可以是以下组合：
- 当前 K 线收阴。
- 上影线明显长于实体。
- 收盘重新回到箱体内部。
- 不允许出现放量实体突破上沿。

### 入场与风控
- 入场：确认 K 线后，下一根 K 线开盘价做空。
- 止损：放在箱体上沿之外，再加 `ATR * stopATRMultiplier` 缓冲。
- 第一止盈：箱体中轴。
- 第二止盈：箱体下沿。
- 首版回测可先简化为固定目标：
- 保守版：到中轴止盈。
- 进取版：到对侧边界止盈。

## 6. 失效条件
- 价格实体连续两根收在箱体外。
- 箱体宽度快速扩张，说明震荡结构被破坏。
- 成交量重新明显放大，并伴随方向性突破。
- 均线重新发散，市场重新进入趋势阶段。

## 7. 参数说明
| 参数 | 含义 | 建议初值 |
| --- | --- | --- |
| `impulseLookback` | 识别前置冲击段的回看窗口 | `30` |
| `minImpulsePercent` | 冲击段最小累计涨跌幅 | `0.03` |
| `minImpulseATRRatio` | 冲击段相对常态波动的强度倍数 | `2.5` |
| `consolidationLookback` | 横盘整理识别窗口 | `12` |
| `maxBoxWidthPercent` | 箱体最大宽度占价格比例 | `0.015` |
| `minBoxWidthPercent` | 箱体最小宽度占价格比例 | `0.003` |
| `consolidationVolumeRatio` | 整理期均量 / 冲击段均量上限 | `0.7` |
| `consolidationATRRatio` | 整理期 ATR / 冲击段 ATR 上限 | `0.7` |
| `minBoundaryTouches` | 上下沿最少测试次数 | `2` |
| `edgeTolerancePercent` | 价格接近边界的容差 | `0.002` |
| `minRejectWickBodyRatio` | 反转确认影线 / 实体最小倍数 | `1.5` |
| `stopATRMultiplier` | 止损 ATR 缓冲倍数 | `0.8` |
| `cooldownBars` | 每次交易后的冷却 K 线数 | `2` |
| `targetMode` | 止盈模式：中轴 / 对侧边界 | `midline` |

## 8. 首版落地约束
- 首版必须先识别“冲击后缩量横盘”，不接受单纯最近 N 根高低点成箱体的简化方案。
- 首版不做趋势追单，只做箱体边界反转。
- 首版不做加仓，不做马丁，不做网格补仓。
- 首版允许多空对称，但同一时刻不重复开同方向新仓。
- 若箱体已经失效，立即停止该箱体下的后续开仓。

## 9. 推荐起步参数
### 偏稳健
- `impulseLookback=30`
- `minImpulsePercent=0.03`
- `minImpulseATRRatio=2.5`
- `consolidationLookback=12`
- `maxBoxWidthPercent=0.012`
- `consolidationVolumeRatio=0.65`
- `consolidationATRRatio=0.65`
- `edgeTolerancePercent=0.0015`
- `minRejectWickBodyRatio=1.8`
- `stopATRMultiplier=0.8`
- `targetMode=midline`

### 偏进取
- `impulseLookback=24`
- `minImpulsePercent=0.025`
- `minImpulseATRRatio=2.0`
- `consolidationLookback=10`
- `maxBoxWidthPercent=0.015`
- `consolidationVolumeRatio=0.75`
- `consolidationATRRatio=0.75`
- `edgeTolerancePercent=0.0025`
- `minRejectWickBodyRatio=1.4`
- `stopATRMultiplier=1.0`
- `targetMode=opposite_edge`

## 10. 后续可扩展项
- 增加“假突破回箱体”专用过滤器。
- 增加成交量异常放大后的禁开仓条件。
- 增加箱体中轴分批止盈和保本止损。
- 增加更高周期方向过滤，例如大周期极强趋势时减少逆势边界单。
